# AJPSC Annuaire

Logiciel de gestion de l'annuaire en ligne des membres de l'Association des Juristes Panthéon-Sorbonne / Universidad Complutense de Madrid

http://www.ajpsc.com

## Installation

1) Téléchargez les fichier dans un dossier géré par le serveur Web, en utilisant un client FTP ou tout autre moyen permettant d'accéder au système de fichier du serveur.
2) Modifier le fichier **includes/config.inc.php** en suivant les instructions de ce fichier pour renseigner les informations nécessaires.
3) **Optionnel.** Placer le dossier de l'annuaire sous protection (.haccess, mot de passe) ; configurer éventuellement une connesion SSL.
